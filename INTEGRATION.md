# توثيق التكامل — عيادة الشفاء

## تشغيل المشروع
```bash
npm install
npm run dev
```
موقع تسجيل الدخول/التسجيل محتاج الـ mock backend شغال في نفس الوقت:
```bash
npm run mock-api
```
(بيشغل json-server على المنفذ 3001 باستخدام Register.json، بالظبط زي ما كان في المشروع الأصلي)

## خريطة الراوتات
| المسار              | المصدر                          | تعديل حصل؟ |
|---------------------|----------------------------------|------------|
| `/` `/login` `/register` | smart-clinic-depi (راوتينج داخلي خاص بيه) | لا |
| `/doctor`           | doctor-dashboard-refactored     | لا |
| `/reception`         | reception-dashboard              | لا |
| `/patient`           | patient-dashboard (pat_.zip)     | لا |

صفحة المريض فيها زرار توليد تقرير طبي PDF (`html2canvas` + `jspdf`) — الباكدجين دول اتضافوا لـ package.json الموحّد زي ما هما بالظبط.

## الملفات الجديدة فقط (طبقة التكامل)
- `src/main.jsx` — نقطة الدخول الموحدة
- `src/router/AppRouter.jsx` — الراوتر المركزي (lazy loading لكل تطبيق)
- `src/apps/*/entry.jsx` — ملف تحميل صغير لكل مشروع (بيستورد نفس الـ CSS اللي كانت في main.jsx القديم بتاعه، وبيصدّر App.jsx الأصلي زي ما هو)
- `package.json`, `vite.config.js`, `index.html`, `eslint.config.js` (من smart-clinic) — موحّدة على مستوى الجذر

كل كومبوننت وCSS ومنطق داخل `src/apps/smart-clinic/`، `src/apps/doctor-dashboard/`، `src/apps/reception-dashboard/` **لم يتغير حرف واحد فيه**.

## ⚠️ المشكلة الوحيدة اللي وقفت التحقق الكامل من البناء (Build)
مجلد `src/apps/smart-clinic/assets/images/` **مش موجود في الملف المرفوع أصلاً** — 12 صورة متستوردة في الكود (`Services.jsx`, `Awards.jsx`, `Hero.jsx`, `Login.jsx`) بس الملفات الفعلية مش موجودة في الـ zip (الموجود بس `assets/unnamed.png`). دي مشكلة أصلية في المشروع الأصلي، ظهرت حتى قبل أي دمج — يعني `npm run build` كان هيفشل في smart-clinic-depi لوحده برضه.

جربت نسخة اختبارية بصور placeholder مؤقتة والباقي كله اشتغل 100% (raوتينج + doctor + reception + تقسيم الـ CSS لكل تطبيق) — يبقى المشكلة الوحيدة هي الصور الناقصة دي بالتحديد، مش أي حاجة في التكامل نفسه.

**محتاج منك:** ترفع الصور الأصلية (أو تقولي أحط صور بديلة مؤقتة لحد ما تجيبها).

## ملاحظات إضافية (بدون أي تعديل، للعلم فقط)
- `Register.json` (قاعدة بيانات تسجيل الدخول الوهمية) اتنقل زي ما هو ومعاه بيانات حقيقية (إيميلات/باسوردات) — راجع الملاحظة الأمنية اللي قلتها في المراجعة السابقة قبل أي رفع عام.
- كل مشروع لسه فيه ملف `main.jsx` الأصلي بتاعه جوه مجلده (مش بيتستخدم دلوقتي، الراوتينج بقى من `src/main.jsx` الجديد بس) — سيبته زي ما هو حسب طلبك إني ميتشالش أي كود موجود.
