# # 🏥 Smart Clinic DEPI

A Smart Clinic Management System built with **React + Vite** as part of the **Digital Egypt Pioneers Initiative (DEPI)**.

## 🚀 Run the Project

### 1. Install dependencies

```bash
npm install
```

### 2. Start the JSON Server

```bash
npx json-server --watch Register.json --port 3001
```

### 3. Start the development server

```bash
npm run dev
```

### 4. Open your browser

```
http://localhost:5173
```

> **Note:** Keep both the JSON Server and the React development server running while using the project.
